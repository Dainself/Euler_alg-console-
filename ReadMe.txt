Данная программа предназначена для запуска на операционных системах Windows 7 SP1, 8, 8.1, 10. Работоспособность на более ранних версиях Windows не гарантирована. Перед запуском рекомендуется убедиться в наличии на оборудовании установленного пакета Microsoft Visual C++ Runtime 2015. Дополнительных библиотек не требуется.
Для использования программы откройте исполняющий файл Euler's algorithm.exe и следуйте приведенным инструкциям.
